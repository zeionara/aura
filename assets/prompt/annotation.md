В качестве исходных данных тебе передается таблица, извлеченная из строительной документации и представленная в виде `json` объекта.
На верхнем уровне этот объект имеет один атрибут - `rows`, содержащий двумерный список объектов с данными из строк таблицы. Каждый элемент списка не первом уровне вложенности соответствует
одной строке исходной таблицы, на втором уровне вложенности - одной ячейке таблицы.
В каждом объекте с данными о ячейке таблицы есть атрибут `id`, позволяющий однозначно идентифицировать ячейку таблицы среди всех остальных элементов документа,
атрибут `cols`, содержащий количество столбцов таблицы, на которые распространяется эта ячейка, атрибут `rows`, в котором указано количество строк, которые охватывает эта ячейка, и
атрибут `text`, в котором представлен текст ячейки.
Обрати внимание, что в таблице могут быть объединенные ячейки. Под объединенной ячейкой имеется в виду такая ячейка, которая нарушает равномерную структуру таблицы из-за того, что
охватывает более одной строки (значение атрибута `rows` объекта, соответствующего содержимому ячейки) или более одного столбца (значение атрибута `cols` объекта, соответствующего содержимому ячейки).
Если ячейка охватывает более одной строки, то в элементы списка, соответствующего верхнеуровневому свойству `rows` объекта таблицы, помещаются заполнители (placeholders) в строках,
включающих эту ячейку, за исключением самой верхней строки - в самой верхней строке находится объект с полным описанием ячейки, который соответствует описанному выше формату. Такие заполнители
характеризуются только одним атрибутом `id`, значение которого совпадает со значением одноименного атрибута объекты с полным описанием содержимого ячейки.
Если ячейа охватывает более одного столбца, то в объекте, соответствующем содержимому этой ячейки, указывается значение атрибута `cols`, равное количеству охватываемых столбцов,
и заполнителей после этой ячейки в той же строке не добавляется.

- Если ты оцениваешь релевантность параграфа таблице числом 0.0, это значит, что текст параграфа не имеет отношения к таблице и описывает совсем другие элементы предметной области;

- Если ты оцениваешь релевантность параграфа таблице числом 1.0, это значит, что текст параграфа необходим для правильного понимания содержимого таблицы. Ты можешь также выставлять любые промежуточные значения релевантности для пограничных случаев в интервале [0; 1] - например, 0.6 или 0.2.
